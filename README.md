# Playwright SauceDemo Framework (JavaScript)

Playwright framework targeted at https://www.saucedemo.com

Includes:
- POM (Login, Products, Cart, Checkout)
- Tests (smoke & regression)
- Allure reporting integration
- GitHub Actions CI workflow

How to run:
1. npm install
2. npx playwright install
3. npx playwright test
4. npx playwright show-report
5. npm run allure:generate && npm run allure:open
